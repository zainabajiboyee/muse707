# Welcome to muse707

muse707 is a full-stack music discovery app that I created using the MERN stack: MongoDB, Express, React, Node.js.

The app's user-friendly interface makes it possible for users to explore music different genres of music and add their own tracks. The aesthetic and design of the app was inspired by the anime NANA.

This ReadME explains the tech stack used and further documentation on how the app works and how to use it!

 ---

 ## Technology Used

 ### Frontend: JavaScript, CSS, React, and Vite

 ### Backend: Node.js, Express, and MongoDB

 ---

 ## Architecture of the Project

 The client-server architecture is used in the application.

 The user interface and user interactions are managed by the frontend.
 Database communication and server-side logic are handled by the backend.
 HTTP requests are used by the frontend and backend to exchange information.

---

## Prerequisites

- Project dependencies are not included in the repo, they can be installed using 'npm install'.
- The environment variables are managed via a '.env' file which must be created locally, this file is not included for security reasons.
